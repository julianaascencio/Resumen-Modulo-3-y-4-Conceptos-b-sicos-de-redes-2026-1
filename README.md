## Módulo 3: Redes inalámbricas y móviles

El módulo 3 del curso Conceptos básicos de redes de Cisco aborda el funcionamiento de las redes inalámbricas y las redes móviles, que permiten la comunicación entre dispositivos sin necesidad de utilizar cables físicos. Estas redes utilizan ondas de radio para transmitir la información y facilitan que equipos como laptops, teléfonos inteligentes, tablets y otros dispositivos puedan conectarse a internet desde diferentes lugares.

Una de las tecnologías más utilizadas en este tipo de redes es Wi-Fi, la cual funciona a través de dispositivos llamados puntos de acceso o routers inalámbricos. Estos equipos permiten que varios dispositivos se conecten a una misma red. Entre las principales ventajas de las redes inalámbricas se encuentran la facilidad de instalación, la flexibilidad para moverse dentro de un espacio y la posibilidad de conectar múltiples dispositivos al mismo tiempo. Sin embargo, también pueden presentar algunas limitaciones como interferencias en la señal, menor estabilidad en comparación con redes cableadas y posibles riesgos de seguridad si no se protegen adecuadamente.

Para que un dispositivo pueda conectarse correctamente a una red Wi-Fi se realizan dos procesos importantes:

* Autenticación: el dispositivo se identifica dentro de la red, normalmente mediante una contraseña.
* Asociación: una vez autenticado, el equipo establece la conexión con el punto de acceso para enviar y recibir datos.

La seguridad es un aspecto fundamental en las redes inalámbricas. Para proteger la información y evitar accesos no autorizados se utilizan diferentes protocolos de seguridad, entre ellos:

* WEP, que fue uno de los primeros métodos de protección pero actualmente es considerado poco seguro.
* WPA y WPA2, que mejoraron el nivel de seguridad de las redes inalámbricas.
* WPA3, que es la versión más reciente y ofrece mayor protección.

Además de las redes Wi-Fi, este módulo también explica las redes móviles, que permiten a los dispositivos conectarse a internet mediante las infraestructuras de telecomunicaciones de los operadores. Entre las tecnologías más conocidas se encuentran 3G, 4G/LTE y 5G, cada una con mejoras en velocidad, cobertura y capacidad de conexión.

## Módulo 4: Construcción de una red doméstica

El módulo 4 se enfoca en explicar cómo crear y organizar una red dentro de un hogar, permitiendo que diferentes dispositivos compartan recursos como la conexión a internet, archivos o dispositivos periféricos. Actualmente, muchas casas cuentan con varios equipos conectados al mismo tiempo, lo que hace necesaria una red doméstica bien configurada.

En una red doméstica es común encontrar dispositivos como:

* Computadores o laptops
* Teléfonos móviles
* Televisores inteligentes
* Consolas de videojuegos
* Impresoras
* Cámaras de seguridad

Para conectar estos dispositivos se utilizan principalmente dos tipos de tecnología de red. La primera es Ethernet, que utiliza cables de red y ofrece mayor estabilidad y velocidad en la transmisión de datos. La segunda es Wi-Fi, que permite conectar equipos sin cables y facilita la movilidad dentro del hogar.

Las redes inalámbricas funcionan gracias a los estándares IEEE 802.11, que definen cómo se transmite la información. Con el tiempo han surgido diferentes versiones que mejoran el rendimiento de la red, como:

* 802.11a
* 802.11b
* 802.11g
* 802.11n
* 802.11ac
* 802.11ax (Wi-Fi 6)

El dispositivo central de una red doméstica es el router, ya que conecta la red del hogar con el proveedor de internet y permite que los dispositivos se comuniquen entre sí. Además, el router se encarga de asignar direcciones IP automáticamente a los equipos mediante el servicio DHCP, proporcionar la señal Wi-Fi y permitir la configuración de seguridad de la red.

Para instalar una red doméstica básica generalmente se siguen algunos pasos sencillos:

1. Conectar el router al módem del proveedor de internet.
2. Acceder a la configuración del router.
3. Definir el nombre de la red inalámbrica (SSID).
4. Establecer una contraseña segura para proteger la red.
5. Conectar los dispositivos del hogar a la red configurada.

En general, este módulo enseña los conceptos fundamentales para comprender cómo funciona una red dentro del hogar y cómo configurarla de forma correcta para que varios dispositivos puedan conectarse y compartir la conexión a internet de manera segura.
